# 🚀 Age and Gender Prediction using Deep Learning

## 📌 Project Overview

This project focuses on predicting the **age and gender of a person from facial images** using **Deep Learning and Computer Vision techniques**.

A Convolutional Neural Network (CNN) model is trained to learn facial patterns and features from images and classify:

* **Gender** → Male / Female
* **Age** → Estimated age group

The implementation is done using **Python, TensorFlow/Keras, OpenCV, and Jupyter Notebook**.

---

# 🎯 Project Objectives

* Detect facial features from images
* Classify **gender (Male/Female)**
* Predict **approximate age**
* Train a **deep learning CNN model** for image classification
* Evaluate model performance
